Semantic controls

A plain HTML form using Axoloth structural utilities for spacing, alignment, controls, and focus states.

Preferences

Tabbed content

Use Arrow Left, Arrow Right, Home, and End while a tab has focus.

Semantic structure

Axoloth links every tab to its panel and keeps only one panel visible.

Roving focus

Arrow keys move through tabs while Home and End jump to the edges.

Framework neutral

The same data attributes work in HTML, React, Vue, Svelte, and Angular.

Manual activation

Arrow keys move focus; Enter or Space activates the panel.

Preview remains selected while focus moves.

Source activates only after confirmation.

Accordion

This group keeps one item open at a time and allows every item to collapse.

Import the CSS, then initialize only the behaviors you use.

Yes. Add data-axo-accordion-multiple to the group.

Arrow Up, Arrow Down, Home, End, Enter, and Space.

Dropdown menu

Open the menu with click, Enter, Space, Arrow Down, or Arrow Up. Navigate its actions with Arrow keys, Home, End, and Escape.

Project

Toast notifications

Toasts announce updates through a live region, pause while focused or hovered, and can be dismissed without a framework.

What this proves

  • Form spacing is handled by axo-form.
  • Responsive fields are handled by axo-form-grid.
  • Controls stay neutral and can be themed by project CSS.
  • Accessibility note: Invalid states include text and ARIA, not color alone.

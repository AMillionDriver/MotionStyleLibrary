# Changelog

## 0.6.0

- Added `initDrawer()` as the first-class drawer controller.
- Added `data-axo-drawer-id`, `data-axo-drawer-toggle`, and `data-axo-drawer-dismiss`.
- Added `axo:drawer-open` and `axo:drawer-close` lifecycle events.

## 0.5.0

- Captured the `0.4.0` package exports, initializers, data attributes, and custom events as a public API baseline.
- Added source-to-registry drift detection and executable deprecation validation.
- Added contract tests for additive changes, aliases, removals, release windows, and package exports.
- Added machine-readable behavior metadata and a migration policy.

## 0.4.0

- Added accessible Dropdown menus with outside-click dismissal and complete keyboard navigation.
- Added programmatic and declarative Toast notifications with regions, timers, pause-on-interaction, limits, actions, and dismissal lifecycle.
- Added framework-neutral exports and combined initialization for both behaviors.

## 0.3.0

- Added accessible Tabs with automatic or manual activation and roving keyboard focus.
- Added single or multiple Accordion groups with collapsible and keyboard navigation modes.
- Added framework-neutral controller APIs and ARIA synchronization for both behaviors.

## 0.2.0

- Added an accessible dialog and modal controller.
- Added nested-layer ordering so Escape and focus trapping affect only the top layer.
- Shared focus, ARIA, backdrop, scroll-lock, and lifecycle behavior with off-canvas drawers.

## 0.1.0

- Added the optional off-canvas sidebar controller.
- Added Escape handling, focus wrapping, focus return, backdrop dismissal, body scroll lock, and ARIA synchronization.
- Added framework-neutral entry points for individual or combined initialization.
